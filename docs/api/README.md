# Receipt API Docs

This directory is the canonical reference for Receipt public surfaces.

## Documents
- [HTTP API](./http.md): all HTTP routes (JSON APIs + HTML/UI routes), request/response details, side effects, and `curl` examples.
- [SSE API](./sse.md): topics, events, and stream subscription endpoints.
- [CLI API](./cli.md): `receipt` commands, flags, output contracts, and examples.
- [SDK API](./sdk.md): exported TypeScript SDK symbols and usage patterns.
- [Streams & Receipts](./streams.md): stream naming, run/branch/sub-run layout, and job lifecycle semantics.
- [Configuration](./config.md): supported environment variables, defaults, and operational impact.
